Mobile Application Development Tool-MIT App Inventor

  1. Name:MIT App Inventor
  2. Version:nb175
  3. Platform:Android platform (iOS support is in the road map) using visual blocks. It is also created by Google, then developed and maintained by MIT. Many different projects are built over App Inventor like MIT spin-out Thunkable, AppyBuilder, Kodular, DroidMaker, Hybro Studio.
  4. MIT App Inventor Website
  5. Developers:Google,MIT Media Lab,MIT Computer Science and Artificial Intelligence Laboratory
  6. Release Date:December 15, 2010
  7. License:Creative Commons Attribution ShareAlike 3.0 Unported, Apache 2.0
  8. Price:Free
  9. Programming Language:Java, Kawa, Scheme
  10. Feature:Access to most of the phone's functionality: phone calls, SMS texting, sensors for location, orientation, and acceleration, text-to-speech and speech recognition, sound, video.
  11. Debugger: Yes
  12. Built-in Emulator: Yes
  13. Machine Requirements in terms of hardware and software: Internet Explorer is not supported. We recommend Chrome or Firefox. Macintosh (with Intel processor): Mac OS X 10.5 or higher Windows: Windows XP, Windows Vista, Windows 7 GNU/Linux: Ubuntu 8 or higher, Debian 5 or higher (Note: GNU/Linux live development is only supported for WiFi connections between computer and Android device.)
  14. IDE: No
